1.token主要目的：在后台中用户的用户名和密码都验证成功之后，向前端会发送一个token，这个token用于在admin
中页面各个接口中安全地获取数据

2.token的生成步骤：

const jwt=required("jsonwbetoken")
const token=jwt.sign({
         _id:user.id
         name:user.name
     },"shdghsagd")
   res.send(token)

3.在前端页面接收token并且储存在localStorage中

localStorage.token=res.data
this.router.push("/)
this.$message({
    type:"success"
    message:"登入成功"
})

4.登入成功之后在admin各个页面的请求接口处，将token返回给后台验证

 axios.interceptors.request,use(function(config){
         if(localStorage.token){
              config.headers.Authorization="Bear"+localStorage 
         }
         return config
},function(err){

})

5.在后端接口处验证token，在接口处使用一个中间件

const jwt=required("jsonwebtoken")
const AdminUser=required("../model/AdminUser)
const assert=required("http-assert")
async (req,res,next)=>{
    const token=String(req.headers.authorization||' ').splice(" ")
    assert(token,422,"请重新登入")
    const tokenData=jwt.verify(token,req.app.get("secret"))
    assert(tokenData._id,422,"请重新登入")
    const  user=await AdminUser.findOne(tokenData._id )
    assert(user,422,"请重新登入")
    await next()
}

6.在后台对错误进行处理

 app.use(async(err,req,res,next)=>{
 res.status(err.statusCode||500).send({
       message:err.message
 })
  })    
}

7.前端页面接收返回的错误信息，并跳转到登入页面

axios.interceptors.response(res={
    return res
},err=>{
    if(err.response.data.message){
        Vue.$protoType.message({
            type:"error",
            message:err.response.data.message
        })
    }
    if(err.response.data.status){
        router.push("/login)
    }
    return Promise.reject(err)
})
